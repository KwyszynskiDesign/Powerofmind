# powerofmind

Statyczna strona wyeksportowana z WordPressa i odtworzona do działania bez backendu.

## Struktura
- site/ — pliki strony (HTML, CSS, JS, assety) ← wrzucisz z pendrive'a
- docs/ — dokumentacja, audity, plany
- .claude/rules/ — reguły pracy dla Claude Code
- .claude/skills/ — skille specjalistyczne dla różnych zadań

## Jak pracować z tym repo
1. Otwórz folder w VS Code.
2. Uruchom Claude Code: `claude`.
3. Claude automatycznie przeczyta CLAUDE.md, reguły i skille.
4. Postępuj zgodnie z planem w docs/MIGRATION_PLAN.md (utworzony przez Claude).

## Deployment
Zobacz docs/DEPLOY.md (do utworzenia po audycie).
