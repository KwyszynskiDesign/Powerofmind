# deployment-rules

- Zakładaj deployment statyczny (GitHub Pages, Netlify, Vercel, hosting klienta).
- Strona ma działać bez PHP i bez bazy danych.
- Wszystkie assety muszą być lokalne w `site/assets/` lub z jawnie wskazanego bezpiecznego CDN.
- Unikaj zależności od środowiska lokalnego w kodzie produkcyjnym.
- Przed deploy: uruchom lokalny serwer i przeklikaj każdą podstronę.
